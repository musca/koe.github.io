Overview

This content slider/Carousel is made to work in all major browsers, but also accessible for users that don't have Java Script enabled.

Can be used in many different projects and is easy to customize. You can use almost any content inside the slides.

jQuery Tools

The slider is based on jQuery Tools library so you can easy incorporate other tools from the library.

If you want to use jQuery code from another project on the page. No problem, jQuery Tools 1.2.5 is based on jQuery 1.4.2.

Features

  • 100% window width.
  • Carrousel with circular mode.
  • Easing plugin support
  • Javascript fallback.
  • Tested & working in IE6-7, Safari, Firefox, Opera.

I created this content slider after not finding any solution to my problem. The problem was that I wanted a full window width content slider that worked on big screens, had javascript fallback, support for all major browsers.

For the sliding effect I used jQuery Tools Scrollable and wrote my own javascript/jQuery Tools resize code to achieve 100% window width.

This is my first real JS/jQuery Tools code I written so it is not perfect. Please give me feedback on things that can be optimized and improved.

You can use this code for free. If you like this work, you can always Flattr me.
